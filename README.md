# 🧪 chemtools

A small collection of utilities for routine tasks in computational chemistry.

> New tools appear only as necessary 🌙

## 📦 Dependencies

+ `pmo` requires `vmd` and `imagemagick`

## 🚀 Installation

Clone the repository and build the tools:

```bash
git clone https://github.com/awilix/chemtools.git
cd chemtools
make
```

Consider adding `bin` to your PATH:

```bash
export PATH=$PATH:/path/to/chemtools/bin
```
